## 🚀 Overview
This project predicts the popularity of news articles using NLP and Machine Learning. By analyzing text features and metadata, the system provides insights into which articles are likely to trend.

## 🧠 Features
- **AI-based Popularity Prediction:** Deep learning and regression models to estimate reach.
- **Proxy Signal Engineering:** Creating custom engagement metrics from raw data.
- **Article Ranking System:** Sorting news articles based on predicted impact.
- **Explainable AI:** Understanding which features (like sentiment or readability) drive popularity.
- **Interactive Streamlit Dashboard:** Real-time visualization of data and model predictions.

## ⚙️ Tech Stack
- **Language:** Python 3.14
- **Analysis:** Pandas, NumPy
- **Machine Learning:** Scikit-learn, Joblib
- **NLP:** TextBlob, TextStat
- **Visualization:** Streamlit, Plotly

## 📊 Workflow
1. **Data Cleaning:** Handling missing values and noise in news datasets.
2. **Feature Engineering:** Extracting linguistic and temporal features.
3. **Proxy Signal Creation:** Generating target labels for "popularity."
4. **Model Training:** Training robust models to recognize viral patterns.
5. **Ranking System:** Implementing logic to highlight top-performing content.
6. **Dashboard Visualization:** Deploying the model for user interaction.
