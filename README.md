# 📰 News Popularity Intelligence System

## 🚀 Overview
This project predicts the popularity of news articles using NLP and Machine Learning.

## 🧠 Features
- AI-based popularity prediction
- Proxy signal engineering
- Article ranking system
- Explainable AI
- Interactive Streamlit dashboard

## ⚙️ Tech Stack
- Python
- Pandas
- Scikit-learn
- NLP (TextBlob, TextStat)
- Streamlit
- Plotly

## 📊 Workflow
1. Data Cleaning
2. Feature Engineering
3. Proxy Signal Creation
4. Model Training
5. Ranking System
6. Dashboard Visualization
